SmartFinance

Overview: SmartFinance is a personal finance manager app designed to help users manage their income, expenses, and financial goals effectively. It includes:

  • Budgeting: Create budgets for categories with monthly or weekly durations, track progress visually, and receive notifications.
  • Expense Tracking: Log expenses with details (amount, category, date, notes), attach receipts, and visualize spending trends.
  • Financial Goal Setting: Set savings goals with deadlines, track progress, and automate savings recommendations.
  • Reports: Export detailed PDF/Excel reports, expense breakdowns, and budget summaries.
  • Security: Biometric authentication and encrypted data storage.

Tech Stack: Kotlin, Jetpack Compose, Clean Architecture (MVVM), Room, Dagger-Hilt, Retrofit, Firebase Auth/Firestore, MPAndroidChart, Espresso/JUnit testing.

Future Enhancements: Recurring transactions, currency conversion, advanced data security (SQLCipher), and more analytics with Firebase Analytics.
